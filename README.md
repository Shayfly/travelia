# Travelia

Travelia is a multilingual travel aggregator website offering the best deals on flights, hotels, car rentals, and transfers worldwide.

## Features

- Search for Flights, Hotels, Car Rentals, and Transfers
- Combined Deals: Flight + Hotel, Flight + Hotel + Car Rental, Flight + Hotel + Transfers
- Supports multiple languages: English, Hebrew, Russian, French, Arabic, Spanish
- User-friendly and responsive design
- Free demo version with example data

## How to Use

1. Browse deals or use the search functionality to find flights, hotels, or packages.
2. Select your preferred options and follow the links to book.
3. Contact us for any questions or support.

## Technologies Used

- HTML5, CSS3, JavaScript
- Public free APIs for demo data integration
- Multilingual support with JavaScript language switching
